## Overview

**BeeAI framework** is an open-source framework designed for building, deploying, and managing AI agent systems. It provides the flexibility and performance needed for scalable AI systems, supporting both Python and TypeSript.

## Key features

| **Feature**              | **Description**                                                                 |
|:-------------------------|:---------------------------------------------------------------------------------|
| **Dual language support** | The framework supports development in **Python** or **TypeScript**, allowing you to work with the language most suitable for your project or team. |
| **Model agnostic**       | You can integrate with a variety of model providers, including **Ollama**, **Groq**, **OpenAI**, **watsonx.ai**, and more, offering flexibility in choosing the model provider that fits your needs. |
| **Seamless integration** | Provides out-of-the-box support for integrating with tools like **LangChain** or using the **Model Context Protocol**. You can also build custom integrations depending on the requirements of your system. |
| **Flexible architecture** | Enables the creation of custom agent workflows and logic patterns using the **workflow engine**, providing the flexibility to design agent behavior based on specific needs. |
| **Production-ready**     | The framework includes features for **caching**, **error handling**, **real-time monitoring**, memory management, state persistence, and secure execution within a sandboxed environment, ensuring itâ€™s ready for production use. |
