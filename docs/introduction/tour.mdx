Welcome to the Grand Tour! You may have already gone through the quickstart and run some sample code. Now it's time to see what you can build with this framework.

This guide shows how to build AI applications, from your first agent to production-ready systems, with sections you can follow step-by-step or skip to what interests you.

## Journey Overview

Here's a quick look at the stages you'll go through and the modules you'll use:

| Stage | What You'll Build | Key Modules |
|-------|------------------|-------------|
| **[Foundation](#foundation)** | Your first intelligent agent | [Agent](https://framework.beeai.dev/modules/agents), [Backend](https://framework.beeai.dev/modules/backend) |
| **[Intelligence](#intelligence)** | Agent with memory, tools & dynamic prompts | [Memory](https://framework.beeai.dev/modules/memory), [Tools](https://framework.beeai.dev/modules/tools), [Templates](https://framework.beeai.dev/modules/templates) |
| **[Orchestration](#orchestration)** | Multi-agent coordination systems | [Workflows](https://framework.beeai.dev/modules/workflows) |
| **[Knowledge](#knowledge)** | Domain-specific knowledge integration | [RAG](https://framework.beeai.dev/modules/rag) |
| **[Production](#production)** | Scalable, monitored systems | [Cache](https://framework.beeai.dev/modules/cache), [Observability](https://framework.beeai.dev/modules/observability), [Serialization](https://framework.beeai.dev/modules/serialization) |
| **[Integration](#integration)** | Connect to the AI ecosystem | [Serve](https://framework.beeai.dev/modules/serve) |
